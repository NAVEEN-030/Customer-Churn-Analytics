# Customer Churn Analytics

## 📌 Project Overview

The **Customer Churn Analytics** project analyzes customer behavior using the Telco Customer Churn dataset. The objective is to identify the factors that influence customer churn and build machine learning models that can predict whether a customer is likely to leave the company.

In addition to predictive modeling, the project includes an interactive Power BI dashboard to visualize customer trends and support business decision-making.

---

## 🎯 Objectives

* Clean and preprocess customer data
* Perform Exploratory Data Analysis (EDA)
* Engineer features for machine learning
* Train and evaluate classification models
* Build an interactive Power BI dashboard
* Generate actionable business insights

## 🤖 Machine Learning Models

The following models were trained and evaluated:

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier

Evaluation metrics include:

* Accuracy
* Precision
* Recall
* F1-Score

---

## 💡 Business Insights

* Customers with month-to-month contracts have the highest churn rate.
* Long-term contracts significantly improve customer retention.
* Customers using electronic check as a payment method are more likely to churn.
* Internet service type influences customer churn behavior.
